## 공통 원칙

- 데이터가 30일 미만이면 분석 전에 경고 표시
- 점수 계산: `합격점수합계 / 전체가중치합계 × 100` → 0-100
- WARNING = 0.5점 (PASS=1, FAIL=0)
- 크리티컬 체크 실패 시 전체 등급 한 단계 강제 하락

## 1. Google Ads 분석 기준

### 카테고리 가중치
| 카테고리 | 가중치 |
|---------|------|
| 전환 추적 | 25% |
| 낭비 소비/네거티브 키워드 | 20% |
| 계정 구조 | 15% |
| 키워드 & 품질 점수 | 15% |
| 광고 & 에셋 | 15% |
| 설정 & 타겟팅 | 10% |

### 핵심 체크 항목 (Critical 우선)

**전환 추적 (Critical)**
- [ ] 전환 액션이 1개 이상 활성화 되어 있는가
- [ ] Enhanced Conversions 활성화 및 검증 완료
- [ ] Google Tag(gtag.js) 모든 페이지에서 정상 실행
- [ ] GA4 연동 및 데이터 정상 수집
- [ ] GA4 + Google Ads 중복 카운팅 없음
- [ ] Consent Mode v2 적용 (EU/EEA 필수, 글로벌 권장)

**낭비 소비 (Critical)**
- [ ] 검색어 보고서 최근 14일 이내 검토
- [ ] 네거티브 키워드 리스트 3개 이상 (주제별: 경쟁사/구직/무료/기타)
- [ ] 계정 레벨 네거티브 리스트 적용
- [ ] 무관한 검색어 소비 5% 미만
- [ ] Broad Match + Manual CPC 조합 없음 (반드시 Smart Bidding과 함께)

**계정 구조 (High)**
- [ ] 브랜드/비브랜드 캠페인 분리
- [ ] 광고그룹당 키워드 15-20개 이내
- [ ] RSA 광고그룹당 활성 광고 3개 이상

**키워드 (High)**
- [ ] 평균 품질점수 7 이상
- [ ] 키워드 중복(같은 키워드 여러 캠페인) 없음

**광고 (Medium)**
- [ ] RSA 헤드라인 8개 이상 (광고그룹당)
- [ ] 광고 강도 "우수" 또는 "좋음"
- [ ] 사이트링크 4개 이상, 콜아웃 4개 이상

**설정 (Medium)**
- [ ] 위치 타겟팅 "현재 위치" (관심 지역 제외)
- [ ] 검색 파트너 네트워크 성과 검토
- [ ] ECPC 사용 중이면 Smart Bidding으로 전환 권고

### Google Ads 점수 기준값
| 지표 | 양호 | 주의 | 위험 |
|------|------|------|------|
| 품질점수(평균) | ≥7 | 5-6 | <5 |
| CTR(검색) | ≥6.66% | 3-6.66% | <3% |
| CVR(검색) | ≥7.52% | 3-7.52% | <3% |
| CPC(검색) | ≤$5.26 | $5.26-8 | >$8 |
| 낭비 소비 비율 | <10% | 10-20% | >20% |
| 광고 강도 | 우수/좋음 | 보통 | 나쁨 |
| 잘못된 클릭 | <5% | 5-10% | >10% |

### Performance Max 추가 체크
- 에셋 그룹 다양성 (텍스트/이미지/동영상/피드)
- 잠재고객 신호 설정 (맞춤 세그먼트, 목록)
- URL 확장 설정 검토
- 브랜드 제외 키워드 적용

## 2. Meta Ads 분석 기준

### 카테고리 가중치
| 카테고리 | 가중치 |
|---------|------|
| 픽셀 / CAPI 건강도 | 30% |
| 크리에이티브 다양성 & 피로도 | 30% |
| 계정 구조 | 20% |
| 잠재고객 & 타겟팅 | 20% |

### Andromeda AI 엔진 핵심 원칙 (2025년 10월~)
- 유사도 점수 60% 초과 광고는 노출 억제
- 유사한 변형 100개 < 진짜 다른 광고 10개
- 크리에이티브 다양성이 현재 #1 성과 레버

### 핵심 체크 항목

**픽셀/CAPI (Critical)**
- [ ] Meta Pixel 모든 페이지 정상 실행
- [ ] Conversions API (CAPI) 활성화 (미적용시 데이터 30-40% 손실)
- [ ] 이벤트 중복 제거 설정 (event_id 매칭, 중복제거율 90% 이상)
- [ ] EMQ(이벤트 매칭 품질) Purchase 이벤트 8.0 이상
- [ ] 표준 이벤트 전체 설정 (ViewContent, AddToCart, Purchase, Lead)
- [ ] 도메인 인증 완료
- [ ] 집계 이벤트 측정(AEM) iOS 설정

**크리에이티브 (Critical)**
- [ ] 활성 크리에이티브 형식 3가지 이상 (이미지/영상/캐러셀/컬렉션)
- [ ] 광고 세트당 크리에이티브 5개 이상
- [ ] CTR이 14일 동안 20% 이상 하락한 크리에이티브 감지 (피로도)
- [ ] 동영상: Stories/Reels 15초 이내, Feed 30초 이내
- [ ] UGC/후기 크리에이티브 테스트 여부
- [ ] 헤드라인 40자 이내, 본문 텍스트 125자 이내

**계정 구조 (High)**
- [ ] 학습 제한 광고 세트 30% 미만 (50% 초과 시 위험)
- [ ] 광고 세트 예산 목표 CPA의 5배 이상
- [ ] 캠페인 수 1-3개로 통합 권장
- [ ] 광고 세트 잠재고객 중복 30% 미만

**잠재고객/타겟팅 (Medium)**
- [ ] 신규고객 빈도(7일): 3.0 미만 (3-5 주의, 5 초과 위험)
- [ ] 리타겟팅 빈도(7일): 8.0 미만 (8-12 주의, 12 초과 위험)
- [ ] 맞춤 잠재고객 활성화 (웹사이트 방문자, 고객 리스트)
- [ ] 유사 잠재고객 복수 시드 크기 테스트 (1%, 3%, 5%)
- [ ] 구매 완료자 신규 캠페인에서 제외

### EMQ 점수 기준
| EMQ | 상태 | 조치 |
|-----|------|------|
| 8.0-10.0 | 우수 | 현재 상태 유지 |
| 6.0-7.9 | 양호 | customer_information 파라미터 추가 |
| 4.0-5.9 | 보통 | CAPI 구현, 데이터 품질 개선 |
| <4.0 | 위험 | CAPI + 향상 매칭 즉시 필요 |

### Meta Ads 점수 기준값
| 지표 | 양호 | 주의 | 위험 |
|------|------|------|------|
| EMQ (Purchase) | ≥8.0 | 6.0-7.9 | <6.0 |
| 중복제거율 | ≥90% | 70-90% | <70% |
| CTR | ≥1.0% | 0.5-1.0% | <0.5% |
| 크리에이티브 형식 | ≥3가지 | 2가지 | 1가지 |
| 세트당 크리에이티브 | ≥5개 | 3-4개 | <3개 |
| 학습 제한 비율 | <30% | 30-50% | >50% |

## 5. Quick Win 판단 기준

다음 조건 모두 충족 시 Quick Win으로 분류:
- severity = "critical" 또는 "high"
- 예상 수정 시간 < 15분

Quick Win 예시:
- Enhanced Conversions 활성화 (5분, Critical)
- 위치 타겟팅 설정 수정 (2분, Critical)
- 네거티브 키워드 리스트 추가 (10분, Critical)
- Advantage+ Placements 활성화 (2분, Medium)
- CAPI 설정 시작 (10분, Critical)
